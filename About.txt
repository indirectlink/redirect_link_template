## Notes

Technical:
- Redirection is done using Javascript, and certain page components depend on Javascript as well. If the user doesn't have these enabled, redirection won't work, and certain aspects will be improperly filled in.

Other:
- This template is not a complete website. It is just a basic redirection page.
